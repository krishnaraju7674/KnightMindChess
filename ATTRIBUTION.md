KnightMind Chess is built with Expo and React Native.

Important open-source packages used by the app include:

- Expo
- React
- React Native
- Expo Router
- chess.js
- @react-native-async-storage/async-storage
- @expo/vector-icons

Before publishing, check package licenses from `package.json` and include any required notices if your final distribution requires them.

## Chess Pieces

The current app renders chess pieces using Unicode chess symbols. It does not include copied third-party chess piece artwork.
