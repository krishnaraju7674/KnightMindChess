# KnightMind Chess

KnightMind Chess is a beginner-friendly Android chess learning app built with Expo and React Native.

## Current Features

- Play vs AI
- Local two-player chess
- Timers: 1m, 3m, 5m, 10m
- AI levels: Easy, Normal, Hard, Pro
- Legal chess rules powered by chess.js
- Castling, en passant, promotion, checkmate, draw
- Premove while AI is thinking
- Coach Hint and AI Guide
- Opening name and opening move count
- Puzzle training with saved progress
- Local rating, leaderboard, profile, friends, and invitations
- Online multiplayer preview counts
- Classic and Midnight themes
- In-app Legal screen
